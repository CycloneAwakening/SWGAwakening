Game Files Installation Folder

Use the Install Folder button to select an installation directory for SWG Awakening game files.

It's recommended to create a folder named SWGAwakening at the root of any hard drive (e.g. C:\SWGAwakening) and install the game files to that location.

Selected Directory:

Suggested Directory: C:\SWGAwakening

Suggested Free Space: 4GB

Star Wars Galaxies Files

To play a Star Wars Galaxies emulator, you must own a copy of Star Wars Galaxies.

Locate your Star Wars Galaxies installation using the SWG Folder button below. You can not use 14 day trial client or any other trial client.


Selected Directory:

Common Install Location:
C:\Program Files (x86)\StarWarsGalaxies

Complete the above, then select Finish to install SWG Awakening.
